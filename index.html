LoadLocalAsset(filepath:string):Instance

FilesEnabled

Loads a Roblox model file (.rbxm, .rbxmx) from the local environment specified by the `filepath` and returns the loaded instance object.

Example Usage

local FilePath="Tool.rbxm" local FileData=-- [[ large string of data ]] repeat task.wait() until NEX_Loaded if not Settings.FilesEnabled then warn("Please turn on FilesEnabled in settings!") end writefile(FilePath, FileData) repeat task.wait() until isfile(FilePath) local Object = LoadLocalAsset(FilePath) Object.Parent = game.Players.LocalPlayer.Backpack print("Loaded and parented:", Object.Name)

Accuracy: NEX Developed

getcustomasset(filepath:string):string

FilesEnabledExternalFilePerms

Uploads a local file (e.g., an image, audio) specified by `filepath` to a temporary content system and returns the resulting rbxassetid:// string URL.

Example Usage

local ImagePath="flower.png" repeat task.wait() until NEX_Loaded local rbxasset = getcustomasset(ImagePath) print("Asset ID:", rbxasset)

Accuracy: NEX Developed

InsertAsset(AssetId:string):Instance

Loads an asset directly from the Roblox website using its `AssetId` and returns the loaded instance object.

Example Usage

local AssetId="123456789" repeat task.wait() until NEX_Loaded local Object = InsertAsset("rbxassetid://" .. AssetId) if Object then Object.Parent = game.Workspace print("Inserted asset:", Object.Name) end

clonefunction(func:function):function

Returns a cloned, exact copy of the inputted Luau `func`. The cloned function is not affected by later calls to `hookfunction` on the original.

Example Usage

repeat task.wait() until NEX_Loaded local original_func = print local cloned_print = clonefunction(original_func) hookfunction(print, function() warn("Hooked!") end) cloned_print("This is not hooked.") print("This IS hooked.")

Accuracy: NEX Developed

setreadonly(table:table, readonly:boolean):null

Sets the read-only status of a table. If `readonly` is true, the table cannot be modified.

Example Usage

repeat task.wait() until NEX_Loaded local myTable = {a = 1, b = 2} setreadonly(myTable, true) myTable.a = 5 print(myTable.a) -- Output is still 1 setreadonly(myTable, false) myTable.a = 10 print(myTable.a) -- Output is 10

Accuracy: NEX Developed

The maximum capability for setidentity.

The permission to enable file functions.

The setting which makes NEX only load when the game is loaded.

The permission to enable HTTP functions.

RobloxScriptRequire:boolean

The permission to enable the custom function Rrequire.

ExternalFilePerms:boolean

The permission to enable ExternalFileFunctions.

The setting to turn on debugging.
