NEX Functions

LoadLocalAsset(filepath:string):Instance

Loads the specified file and returns the loaded object.

Example Usage

local FilePath="Object.rbxm" local FileData=FILEDATA repeat task.wait() until NEX_Loaded if not Settings.FilesEnabled then warn("Please turn on FilesEnabled in settings!") end writefile(FilePath,FileData) repeat task.wait() until isfile(FilePath) --reccommended local Object=LoadLocalAsset(FilePath) --Code using the Object

Accuracy: NEX Developed

getcustomasset(filepath:string, content:string):string

Creates a rbxasset with the provided content and returns it.

Example Usage

local FilePath="flower.png" local FileData=FILEDATA repeat task.wait() until NEX_Loaded if not Settings.FilesEnabled then warn("Please enable FilesEnabled in settings!") end writefile(FilePath,FileData) repeat task.wait() until isfile(FilePath) local rbxasset=getcustomasset(FilePath) --example usage: Classes.Tool.Texture=rbxasset --example return: rbxassetid://aoNeizgs.png

Accuracy: NEX Developed

InsertAsset(AssetId:string):Instance

Loads the AssetId and returns the loaded object.

Example Usage

local FilePath="rbxassetid://ID" repeat task.wait() until NEX_Loaded local Object=LoadLocalAsset(FilePath) --Code using the Object

Accuracy: NEX Approved

ToastNotify(Table{Title,Text,Icon}:table):string

Sends an offical Toast notification with the specified table properties.

Example Usage

repeat task.wait() until NEX_Loaded ToastNotify({ Title="NEX Emulator", Text="NEX", Icon=("rbxassetid://id" or CustomAsset) })

Accuracy: Unaccurate (There are more properties that we don't know of.)

DisconnectFromServer(null):null

Destroys ClientReplicator for effiency, should be used on singleplayer games that do not contain class.Script (class.LocalScript is not affected)

Accuracy: NEX Developed

GetCurrentThread(null):thread

Returns the environment that it had been loaded in's thread.

Example Usage

repeat task.wait() until NEX_Loaded local CurrentThread=GetCurrentThread() --code that uses the current thread for some reason. (You could kill the script when it ends for security and effiency?)

Accuracy: NEX Developed

getrenv(null):table

Returns the unedited Roblox environment usually used inside RobloxScripts.

Accuracy: NEX Developed

clonefunction(function):function

Returns a cloned version of the inputted function. (Unaffected by hookfunction)

Accuracy: NEX Developed

GetRegisteredInstances(null):table

Returns all instances found in Luau's registry. (This does not include non-Instance classed classes such as AuroraScriptService)

Accuracy: NEX Developed
